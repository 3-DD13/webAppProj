# webAppProj

Set up venv: 

```bash
# on Mac 
source venv/bin/activate

# on Windows
venv\Scripts\activate 
```

Install backend req
```bash
pip install -r backend/requirements.txt
```
To run backend
```bash
python backend/app.py
```

FRONTEND/
```bash
cd frontend/
npm install
```
To see local frontend/ changes before commiting:
```bash
npm run dev
```
If changes are made to frontend/:
```bash
npm run build
```
